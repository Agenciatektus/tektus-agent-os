# Domain Framework: Arquitetura de Playbook Operacional

Todo processo mapeado na Tektus deve seguir cinco etapas macro:

1. **Contextualização e Propósito:** O que é esse processo? Qual é a métrica ou KPI que ele impacta?
2. **Setup / Pré-Requisitos:** O que precisa estar em mãos (links, acessos, dados) antes de apertar o play?
3. **Passo a Passo (Step-by-Step):** Ações granulares e diretas. Se tem "se", deve ramificar para uma árvore de decisão.
4. **Árvore de Decisão Avançada:** Se X acontecer, vá para A. Se Y, vá para B. Sempre mapear saídas.
5. **Troubleshooting e Escalada:** Os 3 erros mais comuns que podem acontecer nesse processo e como resolver sem depender do gestor.
